# diox

Language: [English](README.md) | 简体中文

此处是 **diox** 项目的基础仓库。请前往项目各自的路径查看指引。

想要将下列 package 的作为 `dio` 对应 package 的一个分叉进行使用，
请在 `pubspec.yaml` 中配置 `dependency_overrides`，例如：

```yaml
dependency_overrides:
  dio:
    git:
      url: https://github.com/cfug/diox
      path: dio/
      ref: dio
```
